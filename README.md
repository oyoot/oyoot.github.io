# oyoot.github.io
Personal blog

## Creating Posts

Create a new note:

```sh
scripts/new note "Title of the note"
```

Create a new thought:

```sh
scripts/new thought "Title of the thought"
```

Add tags with `--tags`:

```sh
scripts/new note "Title of the note" --tags "tag 1, tag 2"
scripts/new thought "Title of the thought" --tags "tag 1, tag 2"
```
